처방조제프로그램 선택
사용 중인 처방조제프로그램을 선택하세요. 선택한 프로그램에 따라 데이터 파싱 방식이 변경됩니다.
프로그램별 특징:
  • PM3000, 팜플러스20: TXT 파일 형식 (.txt)
  • 유팜: XML 파일 형식 (.xml)
자동 조제 설정
새로운 처방전이 감지되면 자동으로 조제를 시작합니다.
주의: 자동 조제 시 시럽 최대량 설정이 적용됩니다. 초과하는 약물은 자동으로 실패 처리됩니다.
시럽 최대량 설정
시럽조제기의 최대 용량을 설정하세요. 설정된 최대량을 초과하는 약물은 전송되지 않으며, 자동 조제와 수동 조제 모두에 적용됩니다.
설정 가이드:
  • 기본값: 100mL (시럽조제기 표준 용량)
  • ex) 100mL 시럽병에 150mL 분주시 넘칠 수 있음
  • 초과 시: 자동 조제에서는 실패 처리, 수동 조제에서는 경고 메시지
네트워크 스캔
새로운 시럽조제기를 등록하려면 네트워크 스캔을 실행하세요.
